## Sobre o projeto

O **FinAI** foi desenvolvido como projeto de portfólio com foco na evolução para desenvolvimento full stack.

A proposta é oferecer uma interface moderna para controle financeiro pessoal, com análise inteligente e experiência fluida.

O projeto prioriza:

- organização de estado no front-end  
- experiência do usuário (UX)  
- interface moderna inspirada em fintechs  
- interações dinâmicas sem recarregamento  

---

## Funcionalidades

- Dashboard financeiro com visual moderno  
- Análise de gastos com feedback inteligente  
- Sistema de internacionalização (PT, EN, ES)  
- Interface responsiva (desktop e mobile)  
- Sistema de planos (Free / Pro / Anual / Familiar)  
- Atualização dinâmica de dados no front-end  
- Navegação fluida sem reload

## Uso de IA no desenvolvimento

Ferramentas de IA foram utilizadas como **apoio técnico em partes específicas**, principalmente em cenários mais complexos.

A IA contribuiu em:

- estruturação de lógica de interface  
- apoio na resolução de bugs  
- organização de fluxo e navegação  
- sugestões de implementação  

Mais importante que o uso da ferramenta foi o aprendizado obtido, especialmente em:

- gerenciamento de estado no front-end  
- organização de interfaces  
- boas práticas de UX em dashboards
